### 项目目的
中国经济不断发展，国家走向繁荣富强，人们的生活水平不断提高，交通运输网也越来越完善，中国旅游业是否与这些因素有关？探究出中国旅游业发展因素，给予中国旅游发展中肯的意见。

### 数据来源
- 国家统计局
- 世界银行
